# Deep Learning Projects

This repository contains Deep Learning projects developed using TensorFlow and Keras.

---

## Projects Included

### 1. Customer Churn Prediction using ANN

An Artificial Neural Network (ANN) model developed to predict whether a customer is likely to leave a bank based on customer demographics and banking information.

#### Skills Used
- Artificial Neural Networks (ANN)
- TensorFlow
- Keras
- Data Preprocessing
- Classification
- Model Evaluation

---

### 2. Brain Tumor Detection using CNN

A Convolutional Neural Network (CNN) model developed to classify MRI brain images and detect brain tumors.

#### Skills Used
- Convolutional Neural Networks (CNN)
- Computer Vision
- Image Classification
- Medical Image Analysis
- TensorFlow
- Deep Learning
