# Customer Churn Prediction using ANN

## Project Overview

Customer churn prediction is one of the most important applications of machine learning in banking and telecommunications industries.

This project uses an Artificial Neural Network (ANN) to predict whether a customer will leave the bank based on customer information.

---

## Problem Statement

Predict whether a customer will exit the bank or continue using its services.

Target Variable:

- 1 = Customer Left
- 0 = Customer Stayed

---
