# Brain Tumor Detection using CNN

## Project Overview

Brain Tumor Detection is a medical image classification problem that uses MRI images to identify the presence of brain tumors.

This project implements a Convolutional Neural Network (CNN) for image classification.

---

## Problem Statement

Classify MRI brain images into tumor and non-tumor categories using deep learning techniques.

---

## Dataset

Brain MRI Images

Classes:

- Tumor
- No Tumor
